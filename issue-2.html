javascript 对象

javascript 对象

2017-05-02

1.创建直接的实例

var obj = new Object();
obj.propertyName = value;

这个例子创建了对象的一个新实例，并向其添加了一个属性，propertyName是属性名，value是属性值

2.字面量定义

var obj = {
    propertyName: value
}

直接定义了一个变量，变量属性propertyName,值value

3.使用对象构造器

function Person(name, age){
    this.name = name;
    this.age = age;
}

本例使用函数来构造对象,创建 JavaScript 对象实例:

var me = new Person("whl", 30);
